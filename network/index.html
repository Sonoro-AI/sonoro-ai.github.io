Part IV The Network

How Sonoro extends beyond a single home — workforce, partner, ecosystem.

a.The Curam Loop

Assess. Train. Match. Recruit. Develop. Retain. A closed workforce loop nobody else can replicate.

Sonoro's commercial moat is the Curam Loop — a closed workforce cycle linking 140,000 candidate care workers with the assessment frameworks, training pathways, funded apprenticeships and recruitment surfaces inside the platform. Most platforms compete on one stage. Sonoro owns the loop.

i.AssessMarco Standard career assessment runs on the Curam network.
ii.TrainMPW Academy, Learn Care Excel and TLC pathways recommended automatically.
iii.MatchCandidates ranked against operator-specific kitchen success profiles.
iv.RecruitATS receives ranked, prepared candidates with full context.
v.DevelopIn-role learning routed by Ask Marco, scored by the Happiness Algorithm.
vi.RetainPerformance signals feed back into the Marco Standard. The loop closes.

b.Ecosystem integrations

Five integration partners. One orchestration spine. Each one defensible. Together, a category.

Sonoro is not built in isolation. It composes existing assets — MPW Academy content, TLC funded learning, Learn Care Excel modules, the Curam candidate network, and a partner ATS — into a single addressable surface. None of these on their own is a platform. Together, they are.

Sonoro is not software. It is an orchestration layer across a network of specialist platforms.
